BASE CONDITIONS

- agar sum hi 0 ho to it will return empty subset which is possible so it will return true ;

MAIN CONDITION
for(int i=1 ;i<=n ;i++){
    for(int j=1 ;j<=sum ;j++){
        if(arr[i-1]<=j){
            dp[i][j]= dp[i-1][j-arr[i-1]];
